# يومي الرياضي

تطبيق Android بسيط بالعربية لخطة يومية حول تمرين الساعة 1 ظهرًا.

## الوظائف
- منبه استيقاظ 8:00 بصوت Alarm مرتفع.
- باقي المواعيد بنغمة منبه كاملة وليست نغمة إشعار قصيرة.
- جدول الفطار، قبل التمرين، الجيم، الكرياتين، الغداء، السناك، العشاء والنوم.
- قائمة بالمكونات الموجودة في البيت.
- اقتراح الوجبة التالية وإظهار المكونات الناقصة.
- إعادة جدولة المنبهات بعد إعادة تشغيل الهاتف.

## ملاحظات Android
- يجب السماح بـ Exact alarms على Android 12+.
- على بعض هواتف Realme/Oppo/Xiaomi يجب السماح للتطبيق بالعمل في الخلفية وعدم تقييد البطارية.
