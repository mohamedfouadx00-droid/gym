# GYM — الإصدار الاحترافي 2.0

تطبيق أندرويد عربي بواجهة حمراء/سوداء لمساعدة المبتدئ في الجيم على تنظيم اليوم، التدريب، الراحة، الأكل، المخزون، المياه والكرياتين.

## أهم المزايا المنفذة
- واجهة حمراء داكنة مع حركة انتقال بسيطة بين الصفحات.
- الصفحة الرئيسية تتغير حسب الوقت وتعرض توصية اليوم.
- تسجيل: هل ذهبت للجيم اليوم أم لا.
- الجمعة يوم إجازة للجيم تلقائيًا.
- اقتراح تمرين أو راحة بناءً على سجل الأيام المتتالية.
- برنامج جيم للمبتدئ + تمارين منزلية بالمجموعات والتكرارات.
- ملف شخصي: الاسم، العمر، الطول، الوزن، الوزن المستهدف.
- أهداف مياه وبروتين محسوبة مبدئيًا من الوزن.
- مخزن طعام كبير مرتب حسب الفئات، ويشمل العيش المصري البلدي.
- اقتراح وجبة من الأطعمة المتاحة.
- مدربون كمال أجسام بصور مولدة بالذكاء الاصطناعي.
- منبهات قوية للصحوة والوجبات والجيم والنوم.
- إشعارات تحفيز يومية اختيارية.
- متابعة المياه والكرياتين والتزام اليوم.

## بناء APK على GitHub
الملف `.github/workflows/build-apk.yml` يبني APK تلقائيًا عند كل Push إلى main.
بعد نجاح Action: افتح Run ثم Artifacts ثم نزّل `GymDay-APK`.

## ملاحظة مهمة
التوصيات عامة للمبتدئين وليست تشخيصًا طبيًا. الألم الحاد أو الإصابات أو الأمراض المزمنة تحتاج مختصًا.
