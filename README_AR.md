# GYM v2
تطبيق شخصي احترافي لمتابعة الأكل والجيم والمياه والكرياتين والوزن والمخزون.

## النسخة الحالية
- واجهة Dark رياضية Motion
- 5 أقسام: الرئيسية، اليوم، التمرين، مطبخي، المزيد
- منبهات Alarm حقيقية مع Full Screen وSnooze
- إشعارات تشجيع
- مخزون غذائي كبير مرتب بفئات وكميات
- عيش مصري بلدي وبروتين اقتصادي وفواكه وخضروات ومكملات
- خصم تلقائي للكميات عند تسجيل الوجبة
- قائمة مشتريات ذكية للأصناف المنخفضة
- مياه وكرياتين ووزن وتمارين وتقدم يومي
- خطة Full Body للمبتدئ
